# Tienda de Libros de Segunda Mano

Proyecto en desarrollo en PHP utilizando el framework Laravel para el módulo
**Desarrollo Web en Entorno Servidor (DWES) – 2º DAW**.

La aplicación permite la compra y venta de libros de segunda mano, gestionando
usuarios, libros, pedidos y ventas mediante una arquitectura MVC.

---

## Objetivo del proyecto

Desarrollar una aplicación web que permita a una tienda:
- Vender libros de segunda mano a los clientes.
- Comprar libros usados a usuarios registrados.
- Gestionar el catálogo de libros y el stock.
- Administrar usuarios y pedidos desde un panel de administración.

---

## Tecnologías utilizadas

- PHP 8.x
- Laravel
- MySQL
- HTML5 / CSS3
- Bootstrap
- JavaScript
- Git & GitHub

---

## Roles de usuario

- **Usuario**: puede registrarse, comprar libros y vender libros a la tienda.
- **Administrador**: gestiona libros, puede añadir, eliminar, editar libros y controlar el inventario.
